# Markdown 办公编辑器

一个功能强大、设计精美的 Markdown 办公编辑器，支持实时预览、多功能工具箱和莫奈风格主题。

## 功能特性

### 📝 编辑功能
- 实时预览 - 左侧编辑，右侧实时查看渲染效果
- 快速编辑 - 18个快捷按钮插入 Markdown 语法
- 代码高亮 - 支持 25+ 种编程语言
- 搜索功能 - 支持 Ctrl+F 快速搜索
- 自动保存 - 每 3 秒自动保存内容
- 滚动同步 - 编辑器和预览区滚动同步
- 面板调整 - 可拖拽调整面板宽度
- 导入/导出 - 支持导入和导出 Markdown/HTML 文件
- 字数统计 - 实时统计字符数和段落数
- 全屏模式 - 支持 F11 全屏编辑

### 🛠️ 多功能工具箱
按数字键 1-9, 0 或 Q 快速切换：

| 快捷键 | 工具 | 说明 |
|--------|------|------|
| 1 | 计算器 | 日常计算 |
| 2 | 画板 | 自由绘图，支持保存 PNG |
| 3 | 时钟 | 世界时钟（北京、纽约、伦敦、东京） |
| 4 | 待办 | 任务管理，支持勾选完成 |
| 5 | 便签 | 快速记录，可拖拽、可调整大小 |
| 6 | 颜色 | 颜色转换（HEX/RGB/HSL） |
| 7 | 一言 | 随机名言 |
| 8 | 单位 | 长度、重量、温度转换 |
| 9 | 密码 | 密码生成器 |
| 0 | 分析 | 文本分析（字符、单词、行数、段落） |
| Q | 时间戳 | 时间戳与日期互转 |

### 🎨 主题系统
- 5 种莫奈风格配色：印象蓝、睡莲紫、日出橙、日落红、草地绿
- 深色/浅色模式一键切换
- 丰富的动画效果和微交互
- SVG 图标，统一风格

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+B | **粗体** |
| Ctrl+I | *斜体* |
| Ctrl+K | `代码` |
| Ctrl+L | [链接](url) |
| Ctrl+F | 搜索 |
| Ctrl+S | 保存为 HTML |
| Ctrl+Shift+S | 保存为 Markdown |
| Ctrl+Shift+P | 保存进度 |
| Ctrl+Shift+L | 恢复进度 |
| F11 | 全屏模式 |
| Tab | 插入 4 个空格 |

## 快速开始

### 在线使用

直接在浏览器中打开 `index.html` 文件即可使用。

### 本地运行

```bash
# 使用任意 HTTP 服务器
# Python
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 技术栈

- HTML5
- CSS3（CSS 变量、Flexbox、动画）
- 原生 JavaScript（ES6+）
- [marked.js](https://marked.js.org/) - Markdown 解析
- [highlight.js](https://highlightjs.org/) - 代码高亮
- LocalStorage - 数据持久化

## 功能亮点

### 便签功能
- 可自由拖拽到页面任意位置
- 可调整大小
- 支持多个便签
- 自动保存位置和内容
